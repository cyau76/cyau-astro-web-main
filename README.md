# Jaden Au — Audio Engineering & Music Technology Website

This repository contains the source code for my personal website and portfolio.

The website is a short, practical introduction to who I am, what I am building, and the kind of audio and music technology work I do.

## About This Website

This site is where I share:

- My background in live sound and audio production
- Current and past projects
- Notes and resources related to music technology
- Contact information for collaboration and questions

## Website Sections

- **Home** — quick overview and intro
- **About** — personal background, training, and working style
- **Current Work / Past Work / Projects** — ongoing and completed work
- **Resources** — useful references and materials
- **Contact** — ways to reach me

## Tech Stack

- [Astro](https://astro.build/)
- TypeScript
- Markdown/MDX
- Cloudflare Workers + Wrangler

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:4321`.

## Build & Deploy

```bash
npm run build
npm run deploy
```
